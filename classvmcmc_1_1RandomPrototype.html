Detailed Description

template<class EngineT>
class vmcmc::RandomPrototype< EngineT >

A thread-safe interface for STL style pesudo random number generators (PRNG). Each thread accesses it's own static instance of a PRNG, which is realized using the thread_local keyword. In order for each new PRNG instance to be constructed with a new seed, a global variable Random::sSeed (initially to be set with Random::Seed), is incremented and used in each PRNG's constructor.

Template Parameters
EngineTUnderlying random number generator (e.g. std::mt19937).
