Detailed Description

Private class encapsulating Sample chains for parallel tempering.

A ChainConfig describes a set of parallel tempered chains, each chain with its own copy of an adjusted parameter configuration and proposal function. If parallel tempering is not used (only one beta value = 1.0 defined), a ChainConfig contains only one "cold" chain.
