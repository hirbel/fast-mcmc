Detailed Description

This base class models the sampling algorithm of an MCMC, advancing to a new state in the parameter space, accepting or rejecting, and saving that state (subsequently sampling the parameter space). Ideally, this interface should cover single chain algorithms, as well as multi-chain algorithms (like DREAM).
