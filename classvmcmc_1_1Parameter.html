Detailed Description

This class represents a fit parameter of the target function to be sampled from. Its purpose is to instruct the MCMC algorithm at what point in the parameter space (start value) with what step size (error) to start sampling. It also holds optional lower and upper value limits, which correspond to step-function like priors. If the property 'fixed' is set, the sampler should never vary the parameter from its start value.
